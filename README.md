## 🎁 How to Install (For Non-Developers)

If you just want to use the extension and don't want to deal with code or terminals, follow these simple steps:

1. **Download the files:** Click the green **"<> Code"** button at the top of this repository and select **"Download ZIP"**.
2. **Extract the folder:** Find the downloaded `.zip` file on your computer and extract/unzip it. Inside, you will find a folder named `dist`. Drag that `dist` folder onto your Desktop so it's easy to find.
3. **Open Chrome Extensions:** Open Google Chrome, type `chrome://extensions/` into the URL bar, and hit Enter.
4. **Turn on Developer Mode:** Look in the top right corner of the page and toggle the **Developer mode** switch to **ON**.
5. **Load the Extension:** Click the **Load unpacked** button that appears in the top left corner. A file browser will open—select that `dist` folder you put on your Desktop.
6. **You're done!** Pin the extension to your Chrome toolbar, click it to add your Gemini API key, and open any LeetCode problem!

## ✨ Features

* **🧠 Context-Aware AI Tutoring:** Automatically scrapes the active LeetCode problem title and description from the DOM, providing the AI with exact context without requiring copy-pasting.
* **🎨 VS Code-Style Syntax Highlighting:** Code blocks generated by the AI are formatted with standard VS Code Dark themes using `react-syntax-highlighter` for ultimate readability.
* **🔒 Secure BYOK (Bring Your Own Key) Architecture:** User API keys are stored securely on the local machine using Chrome's `chrome.storage.local` API. Keys are never transmitted to a third-party database.
* **⚡ Seamless UI Injection:** A fully responsive, expandable React interface that floats above the LeetCode DOM without disrupting native site functionality or intercepting editor clicks.
* **💬 Markdown Parsing:** Real-time formatting of AI responses, including inline code snippets and structural wrapping to prevent horizontal scrolling issues.

## 🛠️ Tech Stack

* **Frontend:** React 18, JSX, Inline CSS (Flexbox)
* **Build Tool:** Vite
* **AI Engine:** Google Gemini 2.5 Flash API
* **Browser API:** Chrome Extensions API V3 (`activeTab`, `storage`)
* **Libraries:** `react-markdown`, `react-syntax-highlighter`
